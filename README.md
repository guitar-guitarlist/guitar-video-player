# Guitar Video Player

ギター演奏動画やレッスン動画を練習しやすくするブラウザ用プレイヤーです。

## 使い方

- ローカル動画: 「動画を開く」から MP4 などを選択します。
- YouTube動画: YouTubeリンクを貼り付けて「読込」を押します。
- GitHub Pagesで公開すると、YouTube動画の読み込みも使えます。

## 主な機能

- 再生スピード変更
- ピッチ変更（ローカル動画のみ）
- 1 / 3 / 5 / 10秒の戻り・早送り
- A-Bループ
- スマホ対応レイアウト

## ローカル確認

ローカルでYouTube動画も確認したい場合は `start-server.cmd` を実行し、表示されるローカルURLを開きます。
